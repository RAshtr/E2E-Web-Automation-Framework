# Selenium Test Automation Framework

## Project Overview
This project is a basic Selenium automation framework developed using Java, Maven, and TestNG.  
It follows the Page Object Model (POM) design pattern for better structure and maintainability.

## Tech Stack
- Java
- Selenium WebDriver
- Maven
- TestNG
- WebDriverManager
- Git & GitHub

## Project Structure
src/main/java  → Base classes and page classes  
src/test/java  → Test cases  
src/main/resources → Configuration files  
pom.xml → Project dependencies  

## Features
- Maven-based project setup
- Page Object Model implementation
- TestNG test execution
- WebDriverManager integration
- Git version control

## How to Run
1. Clone the repository
2. Open the project in Eclipse or IntelliJ
3. Update Maven dependencies
4. Run tests using:
   - Right click → Run As → Maven test
   OR
   - Using TestNG
